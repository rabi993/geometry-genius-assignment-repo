1. What is DOM and what is the purpose of DOM?

DOM means Document Object Model. It is a programming interface provided by web browsers. Its purpose is to represent and manipulate the structure, content, and style of documents written in markup languages like HTML, XML, and XHTML.
The purposes of the DOM are as follows: Document representation; Dynamic content; Event handling; Cross-platform and Cross-language compatibility; Accessibility.

3. What is event bubble?

Event bubble is a method of event propagation in the HTML DOM API when an event is in an element inside another element, and both elements have registered a handle to that event.
It is a process that starts with the element that triggered the event and then bubbles up to the containing elements in the hierarchy.
In event bubbling, the event is first captured and handled by the innermost element and then propagated to outer elements.

4. What is a callback function and why will you use it?

A callback function is any reference to executable code that is passed as an argument to another piece of code; that code is expected to call back & execute the callback function as part of its job.
I will use callback function. Because, it will make it easier to call the function one after another. It keeps us safe from problems and errors.
